# Idle Tower Defense

## Основной вектор развития

### 1. Боссы и враги
- Добавить боссов в конце каждого тира с уникальными наградами.
- Добавить новые типы врагов.

#### 1.1 Усиления мобов
- Система модификаторов для врагов.
- Усиления можно комбинировать на разных:
  - тирах
  - уровнях
  - испытаниях
  - событиях

---

### 2. Город и здания

Добавить город с системой построек для долгосрочной прокачки.

Каждое здание:
- строится за ресурсы
- открывается только после прохождения определённого тира

#### Здания

- **Золотая шахта**  
  Используется для оффлайн прибыли.

- **Портал**  
  Используется для испытаний.

- **Оружейная**  
  Позволяет экипировать башню предметами.

- **Литейная**  
  Переплавляет одни слитки в другие.

- **Таверна**  
  Даёт особые награды за просмотр рекламы.

- **Лаборатория**  
  Прокачивает различные параметры.

- **Башня**  
  Усложняет забеги, увеличивая награды.

- **Гильдия**  
  Пока не активна.

- **Бойцовский клуб**  
  Ивенты, турниры и соревнования.

- **Дома (декор)**  
  Украшения города, которые также дают небольшие бонусы.

---

### 3. Система рекламы

- **Ускорение исследований**  
  Просмотр рекламы для ускорения (раз в несколько часов).

- **Шахта**  
  Умножение оффлайн прибыли.

- **Испытания**  
  Дополнительные попытки через билеты.
